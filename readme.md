# Learning Azure

# To run terraform

## Step 1 - To initialize Azure provider
Run Terraform init:
```
terraform init
```

## Step 2 - To preview what will be done by Terraform
Run Terraform plan:

```
terraform plan
```

### Step 3 - To create resources
Run terraform apply:

```
terraform apply
```

### Step 4 - To delete all resources
Run terraform destroy:

```
terraform destroy
```
